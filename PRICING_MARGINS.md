## ⚠️ BYOK — Дисклеймер

BYOK поддерживает ключи:
- **Anthropic Claude API** (console.anthropic.com)
- **OpenRouter** (openrouter.ai)

**ВАЖНО:** Использование API ключей регулируется Terms of Service провайдера.
- Anthropic запрещает перепродажу доступа к API третьим лицам без отдельного соглашения
- При BYOK пользователь использует **свой собственный ключ** — это его личный аккаунт, не перепродажа
- Рекомендуется юридически оформить это как "infrastructure hosting for user's own API key"
- Добавить в ToS GetClawster пункт: "При использовании BYOK пользователь несёт ответственность за соблюдение условий своего API-провайдера"
