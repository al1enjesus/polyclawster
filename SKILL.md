# polyclawster-agent

Trade on [Polymarket](https://polymarket.com) prediction markets autonomously via [PolyClawster](https://polyclawster.com).

## How It Works

This skill is a client for the **PolyClawster service** (`polyclawster.com`):

- **Wallet custody:** When you run `setup.js`, a Polygon wallet is created **server-side** by PolyClawster. You receive an agent ID and API key — the private key is stored encrypted on PolyClawster servers, not on your machine. You can deposit USDC to the generated address; those funds are managed by the service.
- **API key:** Stored locally at `~/.polyclawster/config.json`. Used to authorize all trade, portfolio, and sell requests to `polyclawster.com`.
- **Trade routing:** All trades (buy/sell) are executed by `polyclawster.com` on your behalf via Polymarket's CLOB API. The service handles order signing with the server-held key.
- **Signals:** AI trading signals are generated by PolyClawster and fetched via API.

> **Important:** Only deposit funds you are comfortable entrusting to the PolyClawster service. The private key does not leave their servers.

## Quick Start

### 1. Register your agent

```bash
node scripts/setup.js --auto
```

Registers an agent on PolyClawster, gets a wallet address and API key, saves to `~/.polyclawster/config.json`. You start with **$10 demo balance** immediately.

### 2. Browse markets

```bash
node scripts/browse.js "bitcoin"
node scripts/browse.js "election 2026"
node scripts/browse.js "crypto" --min-volume 50000
node scripts/browse.js  # top by volume
```

### 3. Place a trade

```bash
# Demo trade (uses $10 demo balance — no real funds)
node scripts/trade.js --market "bitcoin-reach-100k" --side YES --amount 2 --demo

# Live trade (requires USDC deposit to your agent wallet)
node scripts/trade.js --market "trump-win-2026" --side NO --amount 5
```

### 4. Check balance & positions

```bash
node scripts/balance.js
```

### 5. Close a position

```bash
node scripts/sell.js --list        # View open positions
node scripts/sell.js --bet-id 42   # Close specific position
```

### 6. Auto-trade on signals

```bash
node scripts/auto.js --dry-run                            # Simulate
node scripts/auto.js --demo --min-score 7 --max-bet 5    # Demo
node scripts/auto.js --min-score 8 --max-bet 10 --daily-limit 50  # Live
```

## Link to TMA (optional)

Connect your agent to the PolyClawster Telegram app to monitor it visually:

1. Open PolyClawster TMA → Agents → **"+ Подключить"**
2. Get your claim code (e.g. `PC-A3F7K9`)
3. Run: `node scripts/link.js PC-A3F7K9`


Remove `--demo` when ready for live trading.
